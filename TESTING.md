# قائمة اختبار التطبيق

## اختبار الصفحة الرئيسية
- [ ] تحميل الصفحة بسرعة
- [ ] عرض 28 حرف بشكل صحيح
- [ ] زر "أنشودة الحروف" يعمل
- [ ] زر كتم الصوت يعمل
- [ ] الأزرار الانتقالية تعمل (لوحة التحكم، أناشيد، حول التطبيق)

## اختبار صفحة الحرف
- [ ] تحميل صفحة الحرف
- [ ] التبديل بين التبويبات
- [ ] زر النشيد يعمل
- [ ] الأنشطة التفاعلية تعمل
- [ ] زر الرئيسية يظهر

## اختبار الأنشطة
- [ ] تتبع الحرف
- [ ] عجلة الصوتيات
- [ ] مطابقة الحرف
- [ ] إيجاد الحرف
- [ ] تلوين الصورة
- [ ] لعبة المراجعة

## اختبار الأناشيد
- [ ] أنشيدة الحروف تعمل
- [ ] أناشيد إضافية تظهر تلقائياً
- [ ] تشغيل/إيقاف يعمل

## اختبار لوحة تحكم ولي الأمر
- [ ] عرض الإحصائيات
- [ ] شريط التقدم
- [ ] خريطة الحروف

## اختبار الأجهزة
- [ ] سطح المكتب (Chrome, Firefox, Safari)
- [ ] الهاتف Android
- [ ] الهاتف iPhone
- [ ] التبديل بين الوضع العمودي والأفقي

## اختبار الأداء
- [ ] تحميل الصفحة الرئيسية < 3 ثوانٍ
- [ ] تحميل صفحة الحرف < 2 ثانية
- [ ] سلاسة الحركات

## اختبار PWA
- [ ] التثبيت على الهاتف
- [ ] العمل بدون إنترنت
- [ ] الأيقونة تظهر بشكل صحيح
